# CodonCareAI Frontend

AI-powered medical report analysis platform built with React + Vite.

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Framer Motion

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Optimized for Vercel deployment. Connect your repository and deploy automatically.

## API

Backend API: `https://api.codoncareai.com`
